# ADR 0001: Adopt ADR Process

## Status
Accepted

## Context
The project needs an auditable path for architecture changes while multiple issues are implemented over time.

## Decision
Use numbered ADRs under `docs/adr/` with this template:
- Context
- Decision
- Consequences

Each significant architecture or data contract decision must include an ADR link in the related PR.

## Consequences
- Better traceability for future issue implementation.
- Slight overhead for larger changes.
